# COURSEWORK_DOC — PaperFX (черновик курсовой документации)

Этот документ — каркас “стандартной документации” из требований. Его можно расширять по мере доработки проекта.

---

## 1) Постановка задачи
Разработать сетевую клиент–сервер игру в духе paper.io:
- несколько параллельных игр через комнаты
- авторитетный сервер
- JavaFX GUI на клиенте
- хранение результатов/статистики на стороне сервера в БД
- документированный протокол (в текущей версии — **UDP + JSON**)
