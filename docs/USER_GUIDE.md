# USER_GUIDE — PaperFX (MVP)

## 1) Запуск
1) Поднять БД:
```bash
docker compose up -d
```

2) Запустить сервер:
```bash
./gradlew :server:run
```

3) Запустить клиент:
```bash
./gradlew :client:run
```

---

## 2) Вход в игру
- На экране входа укажите `Username` и `Password`
- Нажмите:
  - **Register** — регистрация
  - **Login** — вход
- Сервер по умолчанию: `localhost:7777` (в текущем клиенте адрес фиксирован)

После успешного входа вы автоматически попадёте в комнату `MAIN`.

---

## 3) Управление
- Движение: **WASD** или **стрелки**
- Диагональ не поддерживается
- Чат:
  - введите сообщение и нажмите **Enter**
  - быстрые сообщения (quick chat): **Q / E / R** (отправляют заранее заданные фразы)

---

## 4) Что отображается
- Слева: игровое поле (территории, игроки, след)
- Справа:
  - Leaderboard (текущая территория игроков комнаты)
  - Chat (сообщения комнаты)

---

## 5) Известные ограничения MVP
- Нет отдельного лобби и списка комнат в UI (хотя сервер умеет `create_room/join_room`)
- Нет матчей с таймером/победителем — игра идёт непрерывно
