# PaperFX v0.5

В этой версии территория должна **закрашиваться целиком**, а не только путь.

Что изменено по сравнению с прошлой версией:
- Сервер: заливка после замыкания — **global flood-fill**, а не bbox.
- Сервер: при движении записываются **все пересечённые клетки** (Bresenham), чтобы не было разрывов в следе.
- Клиент: территория рисуется по `owners[]` (сплошной цвет), след — полупрозрачным поверх.

Запуск:
```bash
gradle :server:run
gradle :client:run
```
